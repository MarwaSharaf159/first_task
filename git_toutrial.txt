                                           ---------------------------------------------- Git -------------------------------------------------------------------


to initialize git repo and manage project in this folder

-- git init

to know status for this folder (ex to know untracked files)

-- git status

1-to add all files in the folder to git and this files move to staging area prepare to commit

-- git add .

to add specific file in folder to git 

-- git add fileName

2-to save changes in files to git when use commit ,automatically created master branch and this commit saved inside the master branch inside our repo in this folder

-- git commit -m "any comment"

to show branch that save this commit 

-- git branch

when change any thing in file (ex more lines in file more code) then repeat 1,2

-- git add .
-- git commit -m "any comment"
